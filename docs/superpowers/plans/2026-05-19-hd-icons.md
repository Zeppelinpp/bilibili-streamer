# HD Icons Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace all low-resolution bitmap app icons with a single SVG source, generating crisp icons for all platforms and display densities.

**Architecture:** Create one `source.svg` and use Tauri CLI `icon` command to regenerate the entire bundle. Use ImageMagick to produce a grayscale tray icon from the same SVG.

**Tech Stack:** Tauri CLI, ImageMagick, SVG

### Task 1: Create SVG source file

**Files:**
- Create: `src-tauri/icons/source.svg`

- [ ] **Step 1: Write source.svg**

  The icon is a Bilibili TV shape: rounded-rect body, two antenna stubs, transparent screen cutout, and two vertical rounded-rect eyes inside the screen. Color is `#0097CB` (extracted from existing icon.png histogram).

  ```svg
  <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 1024 1024">
    <defs>
      <mask id="screen">
        <rect width="1024" height="1024" fill="white"/>
        <rect x="160" y="280" width="704" height="580" rx="80" ry="80" fill="black"/>
      </mask>
    </defs>
    <g mask="url(#screen)">
      <rect x="80" y="120" width="864" height="820" rx="160" ry="160" fill="#0097CB"/>
      <rect x="240" y="40" width="80" height="200" rx="40" ry="40" fill="#0097CB" transform="rotate(-20 280 140)"/>
      <rect x="704" y="40" width="80" height="200" rx="40" ry="40" fill="#0097CB" transform="rotate(20 744 140)"/>
    </g>
    <rect x="300" y="420" width="120" height="300" rx="50" ry="50" fill="#0097CB"/>
    <rect x="604" y="420" width="120" height="300" rx="50" ry="50" fill="#0097CB"/>
  </svg>
  ```

- [ ] **Step 2: Verify SVG renders correctly**

  Convert to PNG at 512x512 and visually compare with the original:
  ```bash
  magick src-tauri/icons/source.svg -resize 512x512 /tmp/source-preview.png
  ```
  Open `/tmp/source-preview.png` and confirm it closely matches the shape of the existing `icon.png`.

- [ ] **Step 3: Commit**
  ```bash
  git add src-tauri/icons/source.svg
  git commit -m "assets: add HD SVG icon source"
  ```

### Task 3: Generate HD tray icon

**Files:**
- Modify: `src-tauri/icons/tray-icon-macos.png`

- [ ] **Step 1: Convert SVG to grayscale PNG at 128x128**
  ```bash
  magick src-tauri/icons/source.svg \
    -colorspace Gray \
    -alpha on \
    -resize 128x128 \
    src-tauri/icons/tray-icon-macos.png
  ```
  Expected: File is created, approximately 2-5KB, dimensions 128x128.

- [ ] **Step 2: Verify tray icon**
  ```bash
  file src-tauri/icons/tray-icon-macos.png
  magick identify src-tauri/icons/tray-icon-macos.png
  ```
  Expected: `PNG image data, 128 x 128, 8-bit gray+alpha`.

- [ ] **Step 3: Commit**
  ```bash
  git add src-tauri/icons/tray-icon-macos.png
  git commit -m "assets: regenerate HD tray icon from SVG source"
  ```
